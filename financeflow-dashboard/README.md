# 💸 FinanceFlow Dashboard

Dashboard financeiro pessoal estilo Power BI, desenvolvido com HTML, CSS e JavaScript puro.

## 📊 Funcionalidades

- **KPI Cards** — Saldo total, total de entradas, total de saídas e número de transações
- **Gráfico de Barras** — Entradas vs Saídas comparadas por categoria
- **Gráfico de Rosca** — Distribuição percentual das saídas por categoria
- **Gráfico de Linha** — Evolução do saldo ao longo das transações
- **Categorias** — Alimentação, Transporte, Moradia, Saúde, Lazer, Educação, Salário e Outros
- **Histórico** — Tabela com todas as transações registradas
- **Responsivo** — Funciona em desktop e mobile

## 🚀 Como usar

### Opção 1 — Abrir direto no navegador
Basta abrir o arquivo `index.html` no navegador. Não precisa de servidor.

### Opção 2 — VS Code com Live Server
1. Instale a extensão **Live Server** no VS Code
2. Clique com botão direito em `index.html`
3. Selecione **"Open with Live Server"**
